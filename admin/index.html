Need an access token?
  1. Open github.com/settings/personal-access-tokens/new
  2. Name it anything, set expiration to 1 year
  3. Repository access → Only select repositories → pick your portfolio repo
  4. Permissions → Repository permissions: Contents = Read and write, Actions = Read-only
  5. Generate token, then click Log in above and paste it
