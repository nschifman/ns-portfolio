Sign in

Paste a GitHub personal access token to manage the site. The token never leaves this browser — it's stored locally on this device only.

How to create a token (about 1 minute)
  1. Open github.com/settings/personal-access-tokens/new
  2. Name it anything (e.g. "portfolio admin"), set expiration to 1 year
  3. Under Repository access choose Only select repositories → pick your portfolio repo
  4. Under Permissions → Repository permissions, set Contents to Read and write and Actions to Read-only
  5. Click Generate token and paste it above
